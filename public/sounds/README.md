# Sound Effects

Place audio files here (.mp3, .wav, or .ogg format):

- **achievement.mp3** - Badge unlocked sound (triumphant, upbeat)
- **level-up.mp3** - Level up sound (ascending melody)
- **badge.mp3** - Badge earned sound (quick positive chime)
- **notification.mp3** - Generic notification (subtle ping)
- **message.mp3** - New message sound (friendly bloop)
- **success.mp3** - Success action sound (positive confirmation)
- **error.mp3** - Error sound (gentle negative beep)
- **warning.mp3** - Warning sound (attention-grabbing but not harsh)

## Where to find free sound effects:
- https://freesound.org/ (CC0 and CC-BY licensed sounds)
- https://mixkit.co/free-sound-effects/ (Free for commercial use)
- https://www.zapsplat.com/ (Free with attribution)

## Audio specifications:
- Format: MP3 or OGG recommended for web
- Duration: 0.5-2 seconds (keep it short!)
- Sample rate: 44.1kHz
- Bit rate: 128-192 kbps
- Volume: Normalize to -3dB to avoid clipping
